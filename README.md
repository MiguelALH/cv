## Requirements

- `texlive-xetex`

Ubuntu/Debian

```shell
sudo apt install texlive-latex-extra
sudo apt install texlive-fonts-extra
```

- Fonts: Font Awesome, Hack Nerd Font

## Build

`xelatex file-cv.tex`

Run the command multiple times on the first build.
